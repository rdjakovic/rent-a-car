Rent-a-car sample Spring MVC application
========================================

This project is Spring MVC + Bootstrap + Tiles application with pure Java configuration

- Spring MVC 4.3.3
- Hibernate 4.3.4 (as JPA provider)
- Spring Data JPA 1.4.5
- MySQL 5.1.30
- Apache Tiles 3.0.5
- JSTL 1.2
- Spring Forms JSTL
- Bootstrap 3.2.0
- Log4j
- Spring Security 4.x
